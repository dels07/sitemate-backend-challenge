# Sitemate Backend Challenge

## Server

- Created using node, express, typescript
- Using array as data store
- by default running on http://localhost:9000

### How to Run

```
cd server
npm install
cp .env.sample .env
npm run dev
```

## Client

- Created using node, react, shadcn ui

### How to Run

```
cd client
npm install
npm run dev
```
